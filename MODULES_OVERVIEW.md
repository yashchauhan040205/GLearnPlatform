## 🎮 XP & Level-Up System

### How It Works:
1. **Complete a Module** → Earn 50-200 XP (based on difficulty)
2. **XP Accumulates** → Your total XP increases
3. **Level Up!** → Every 1000 XP = +1 Level
4. **Leaderboard Updates** → Real-time ranking changes

### Level Formula:
```
Level = (Total XP ÷ 1000) + 1
```

### Examples:
- **500 XP** = Level 1
- **1,500 XP** = Level 2
- **5,000 XP** = Level 6
- **10,000 XP** = Level 11
- **25,000 XP** = Level 26
