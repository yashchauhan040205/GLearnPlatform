## 🎮 Quick XP Earning Guide

### Single Chapter (Easiest)
```
Click Chapter → Read Content → Mark Complete
Time: 15-45 minutes
Reward: +50-150 XP
```

### Complete Quiz (Medium)
```
Take Quiz → Answer Questions → Submit
Time: 10-20 minutes
Reward: +100-200 XP (if pass 70%+)
```

### Complete Course (Best)
```
Finish All Chapters → Course Complete
Time: 3-10 hours (total)
Reward: +2000-4000 XP (with bonus!)
```

### Unlock Badge (Automatic)
```
Meet Criteria → Auto-Awarded
Examples:
- Complete 1 chapter → "First Steps" (+100 XP)
- Earn 500 XP → "Knowledge Seeker" (+150 XP)
- 7-day streak → "Week Warrior" (+300 XP)
```
