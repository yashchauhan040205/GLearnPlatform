### Step 4: Level Up System

**How Leveling Works:**
```
Your Level = floor(Total XP / 1000) + 1

Examples:
- 0-999 XP    = Level 1
- 1000-1999 XP = Level 2
- 2000-2999 XP = Level 3
- 5000-5999 XP = Level 6
- 10000+ XP    = Level 11+
```

**What Happens When You Level Up:**
- 🎊 Animated notification
- 📊 New level badge appears in your profile
- 🏆 Title changes (Novice → Explorer → Scholar → Expert → Master)
- ⬆️ Higher position on leaderboard

**XP Sources:**
1. **Chapter Completion**: Main source (50-150 XP each)
2. **Quiz Completion**: Pass quizzes for 100-200 XP
3. **Course Completion**: Bonus 500-3000 XP
4. **Badge Unlocks**: Each badge gives bonus XP
5. **Daily Streaks**: Consistency multipliers
