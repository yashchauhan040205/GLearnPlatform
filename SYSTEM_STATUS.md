## 🚀 How It All Works

### XP System:
- Complete module → Earn XP (50-200) depending on difficulty
- XP accumulates in student profile
- Level auto-calculated: `Level = Math.floor(XP / 1000) + 1`

### Leaderboard:
- Real-time updates via Socket.io
- Every module completion broadcasts `leaderboard:update`
- All connected clients see updated rankings instantly
- Ranked by total XP (highest first)

### Course Progress:
- Track X/Y modules completed per course
- Progress percentage shown
- Completion triggers bonus XP award
- Badge checks run after each completion

### Module Flow:
1. Student clicks module
2. Views content
3. Clicks "Mark Complete"
4. Backend:
   - Adds module to `user.completedLessons`
   - Adds XP to `user.xp`
   - Runs `user.calculateLevel()`
   - Checks for course completion
   - Checks for badge eligibility
   - Saves user
5. Frontend:
   - Shows XP earned toast
   - Shows level-up if occurred
   - Shows badge unlock if earned
   - Updates dashboard stats
   - Animates progress bars
